Contributing to BoronEngine

Thank you for your interest in contributing to BoronEngine!

Reporting Bugs

If you find a bug, you can:

- Open a GitHub issue
- Create a bug report in the Discord server

Please include:

- A description of the problem
- Steps to reproduce it
- Screenshots or logs if available

Suggesting Features

Feature requests are welcome.

You can:

- Open a GitHub issue
- Post your idea in the Discord server

Please explain:

- What you would like to add
- Why it would be useful
- Possible implementation ideas

Pull Requests

Before submitting a pull request:

1. Make sure the project builds successfully.
2. Follow the existing code style.
3. Test your changes and make sure they work on all backends.

Questions

If you have any questions, feel free to ask in the Discord server or open a GitHub discussion.

Thank you for helping improve BoronEngine!
